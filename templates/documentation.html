* Digital Signature Web App – Documentation *

* Project Aim

This project implements a secure Digital Signature System using Flask and RSA cryptography. Users can upload PDF files, sign them digitally, verify signatures, manage logs, and download signed files.

* Technologies Used

  • Python
  • Flask Web Framework
  • RSA Cryptography (cryptography library)
  • HTML & CSS
  • Frontend UI with pink Glow

* Features

  • User Login System
  • PDF Upload & Digital Signing
  • Signature Verification
  • View Logs
  • View Signed Files
  • Delete Files
  • pink UI Dashboard

* How Signing Works

1. User uploads a PDF
2. PDF converted into bytes
3. Private key signs hash using RSA
4. Signature (.sig) file generated

* How Verification Works

1. User uploads PDF + SIG file
2. System reads PDF bytes
3. Public key verifies RSA signature
4. Validity shown on screen
